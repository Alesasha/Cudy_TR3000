# Инструкция для Димы: Linux Agent

Архив для установки: `DC_via_Cudy-linux-prod.zip`.

## Установка

1. Распаковать архив в отдельную папку.
2. Открыть терминал в этой папке.
3. Выполнить:

```bash
chmod +x *.sh
./one_click_install.sh
```

Если попросит пароль `sudo`, ввести пароль Linux-пользователя.

Установщик делает несколько действий:

- восстанавливает прямой интернет-маршрут перед началом;
- при необходимости пытается скачать `sing-box` в `./runtime/sing-box`;
- запускает один smoke-тест;
- ставит и запускает systemd-сервис `cudy-managed-agent.service`;
- в конце показывает диагностический вывод `./status.sh`.

Если установка падает, `one_click_install.sh` автоматически печатает `status.sh`.
Этот вывод нужно отправить целиком.

## Проверка

После установки выполнить:

```bash
./status.sh
```

Нормальные признаки:

- в блоке `control` есть ответ `{"ok": true}`;
- сервис `cudy-managed-agent.service` активен;
- в `managed transports` есть только реально нужные транспорты;
- в `log tail` есть строки `cycle applied`, `routes applied` или `probe jobs processed`;
- интернет не пропадает при включенной AmneziaVPN/агенте.

## Если что-то не работает

Для диагностики достаточно прислать полный вывод:

```bash
./status.sh
```

Скрипт ничего не меняет. Он показывает маршруты, DNS, firewall-подсказки,
активные VPN/TUN-интерфейсы, состояние control tunnel и последние строки лога.

## Откат

Если интернет сломался и нужно только вернуть прямой маршрут:

```bash
sudo ./restore_direct.sh
```

Если нужно полностью убрать агент:

```bash
sudo ./uninstall_systemd.sh
```

## Без скачивания sing-box

Если скачивание бинарников запрещено или интернет нестабилен, заранее положить
`sing-box` в `runtime/sing-box` и запустить:

```bash
AUTO_INSTALL_SINGBOX=0 ./one_click_install.sh
```
