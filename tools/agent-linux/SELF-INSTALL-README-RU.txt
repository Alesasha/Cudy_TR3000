Установка iSashaVPN / Cudy agent

1. Распакуйте этот архив в отдельную папку.

2. Откройте терминал в этой папке и выполните:

   bash ./DC_via_Cudy-linux-self-install.sh

Скрипт сам остановит старый агент, удалит старые файлы из этой папки,
распакует свежий пакет, установит агент, запустит его и выполнит тесты.

Если интернет сломался после тестов, выполните:

   sudo ./restore_direct.sh
